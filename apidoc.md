# checkMe API Documentation

## Endpoints :

List of available endpoints:

- `GET /medicalrecord`
- `GET /symptomrecords`
- `POST /symptomcheck`
- `POST /symptoms`
- `POST /disease`
- `GET /registereddiseases`
- `POST /gsearch`
- `GET /numberdata`
